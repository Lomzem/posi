# posi

🔥 blazingly fast app to calculate maximum share size.

## Hotkeys

- `b`: select `Balance`
- `e`: select `Entry`
- `s`: select `Stop`
- `x`: select `Exit`
- `r`: select `R Multiple`
- `Control + C`: copy maximum share size

## Calculation

- The maximum share size is calculated by the following formula:

  ```
  (balance * rmultiple) / (entry - stop)
  ```
